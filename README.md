# Online Book Borrowing Platform

A seamless and modern web application designed to digitize the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally. The platform prioritizes security and performance using BetterAuth, Next.js, and MongoDB

---

### Project purpose

- Browse a collection of books
- Search and filter by category
- View detailed book information (authenticated users only)
- Borrow books (authenticated users only)
- Manage their profile and update information (authenticated users only)

## Key Features

### Authentication

- Secure Access: User Registration and Login with full form validation.

- Social Login: Google Social Login integration via BetterAuth.

- Conditional UI: Navbar dynamically updates based on authentication state (Login/Logout/Profile).

### Book Exploration

- Search Functionality: Find books instantly by title using the integrated search bar.

- Category Filtering: Functional sidebar to filter collections by Story, Tech, and Science.

- Featured Books: Showcases the top 4 picks directly on the homepage.

### Interactive Elements

- Modern UI: Features a dynamic Banner and a Scrolling Marquee for new arrivals.

- Social Proof: Custom Stats and Testimonial sections built with unique designs.

- Borrowing System: Private route for book details with one-click borrowing and toast feedback.

### Profile Management

- Private Profile: Dedicated page for logged-in users to view their account details.

- Update Info: Feature to update Name and Profile Image URL seamlessly.
